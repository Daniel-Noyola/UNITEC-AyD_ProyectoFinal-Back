# Mapa de inseguridad
Este es el proyecto sobre un mapa de inseguridad de la materia Analisis y Diseño de Software.

## Comandos para desarrollo

### Con php
```bash
php -S localhost:3000 -t public
```

### Con wamp
(Remplazar X.Y.Z con la version de php instalado)
```bash
C:\\wamp64\\bin\\php\\phpX.Y.Z\\php -S localhost:3000 -t public
```

Se debe instalar [composer](https://getcomposer.org/download/) para poder instalar las dependencias.

Correr el siguiente comando al descargar el repositorio para descargar las dependencias una vez instalado composer.
```bash
composer install
```
